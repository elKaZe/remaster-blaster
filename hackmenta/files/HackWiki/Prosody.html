Prosody


Es un servidor XMPP multiplataforma escrito en Lua (lenguaje de programación).

Contenido

  • 1 Instalación
  • 2 Configurar
    • 2.1 Certificado
    • 2.2 Configurar el servidor
    • 2.3 Generar cuentas

Instalación

sudo aptitude install prosody

Configurar

Antes de configurar el servidor tenemos que tener en cuenta que necesitamos un DNS apuntando a nuestra computadora, como alternativa a internet podemos usar Ciboulette con este servicio solo prodrás chatear con personas dentro de la misma red, o sea que hallan configurado el prosody en sus maquinas.

Para ciboulette hay un metodo simplificado para la instalación y configuración del prosody

Certificado

openssl req -new -x509 -days 365 -nodes -out "/etc/prosody/certs/dominio.net.crt" -newkey rsa:4096 -keyout "/etc/prosody/certs/dominio.net.key"
