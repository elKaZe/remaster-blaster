Proxy transparente

El proxy transparente está configurado con Polipo para hacer caché. Como polipo no tiene soporte para proxy transparente (le rompen las bolas al autor), tinyproxy está configurado para aceptar las conexiones que le reenvía el router con MallaCritica. Estos son los archivos de configuración.
