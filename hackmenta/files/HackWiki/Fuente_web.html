Fuente web

Una fuente web, canal web o feed (atom, rss o rdf) es un medio de re-difusión de contenido web. Se utiliza para suministrar información actualizada frecuentemente a sus suscriptores. En su jerga, cuando una página web "re-difunde" su contenido mediante una fuente web, los internautas pueden "suscribirse" a ella para estar informados de sus novedades.

Lector de noticias

Los lectores de noticias sirven para recibir las novedades de las paginas que publican cosas interesantes sin la necesidad de estar "visitándolas" todos el tiempo, el lector descarga los datos directamente en el cliente y vemos las noticias en nuestras computadoras. Son excelentes para reemplazar los noticieros (televisivos) y recibir noticias de fuentes no convencionales: centros culturales, diario zonales, blogs de opinion, diarios alternativos, canales de videos, etc.

Brief es un cliente de noticias para Mozilla Firefox

Como hacer que Firerfox nos avise de las fuentes web

En las ultimas versiones esto no viene por defecto y hay que agregarlo a mano.

Botón derecho del mouse sobre un espacio en blanco en la barra del navegador. E ir a la opción de personalizar.

Seleccionar el logo de subscribirse y presionar el botón de agregar a la barra.

Como subscribirse a una fuente web

Al visitar una pagina web con publicación de noticias uno puede subscribirse haciendo clic en el logo

Al leer una fuente web uno puede subscribirse a ella para futuras lecturas.

Para que Bief tome las novedades hay que elegir la carpeta Canales web subscriptos

Exportar e Importar fuentes

Si queres compartir o recibir las paginas a las que están sindicalizadas un amigo tan solo tenes que pedirle que las exporte e importar el archivo .OMPL. Estos están disponibles en las preferencias del Brief.

Descarga el ejemplo con fuentes de noticias nacionales e internacionales, espacios autogestionados y proyectos de cultura libre.

Una vez descargado presiona en importar... y selecciona el archivo fuentesweb.opml (por lo general este se descarga en la carpeta Descargas). Luego de eso vamos a tener muchas fuentes de información adicional en nuestro Brief.

Piratoteca

Podemos buscar películas, historietas, musica, libros en Torrentz con fuente web, descargarlas con Transmission y verlas en nuestras computadoras con Mplayer, VLC, Comix, FBreader, etc.

Publicación de noticias

Nota: Es muy importante que tu blog tenga una fuente web para sus novedades, sobre todo tiene que estar hecho este modo que puedas transmitir las novedades fácilmente sin necesitas de intermediarios: facebook, google, correo de novedades, etc.

Una buena forma de tener esto bien ordenado es un usar wordpress y escribir las novedades como artículos del blog y NO como paginas; las paginas son estáticas y no aparecen en la fuente web.

En ajustes, lectura y donde dice Mostrar, para cada entrada en el feed como Texto completo

Otra cosa que permiten las fuentes web es replicar noticias de amigos, colegas u otras organizaciones, así si tenemos un grupo de amigos que tiene feed en sus blog, podemos replicarlas en el nuestro. Esto es muy útil para organizaciones sociales: bibliotecas, centros culturales, cines, atenemos, etc.
