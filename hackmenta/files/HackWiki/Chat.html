Chat


IRC es un protocolo en tiempo real basado en texto, que permite debates entre dos o mas personas.

Actualmente estamos usando el canal ircs://irc.hackcoop.com.ar:6697/#lab o si estás en LibreVPN podés conectarte a ircs://naven.local:6697/#lab.

En la web

También podes usar la Interfaz web.

Programas

Podes instalar Pidgin con su complemento para OTR.

Para usar el chat desde el Mozilla Firefox podes agregar Chatzilla + español

En OS X (10.5 +) podes instalar Adium que tiene soporte para OTR. configurar IRC con Adium


Configurar el canal de IRC en Pidgin


El gestor de cuentas


Configurar el servidor irc.hackcoop.com.ar


Es necesario cambiar el puerto a 6697 y habilitar SSL




Menú Amigos > Agregar un canal. Con el botón Lista de salas podemos ver qué otras salas/canales/chats hay en el servidor (aparte de #lab, por ejemplo)


Una vez agregado aparece en la lista de contactos.
