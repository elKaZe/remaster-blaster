Hurricane Electric (DNS dinámico)


https://dns.he.net es un servicio privado que ofrece administración de DNS. Tiene un "Términos y Condiciones" y un límite de 50 dominios por cuenta. Además no soporta dominios genéricos (*.dominio.org).

Si preferís un servicio autónomo, mirá Owns :)

Crear un registro

Acá hay que prestar atención a los registros NS. Estos son los servidores de he.net donde hay que delegar nuestro dominio. Esto se hace desde la administración del registrador de dominios que usaste (https://nic.ar, etc.).

Si no agregás los NS de he.net en tu dominio nunca van a andar!!



Para que se pueda ir de un dominio a una dirección IP, hay que crear un registro A. Este es el registro más básico de un dominio.

Para este registro hay que indicar el dominio para el que se va a crear (el dominio que acabamos de crear o un subdominio ("www" también es un subdominio)), la dirección IP actual (se puede averiguar yendo a http://ns.librevpn.com.ar) y marcar la opción "Enable entry for dynamic DNS".

Zonas en bruto

Un dominio y sus registros se llama "zona". Si vas a migrar de proveedor, con la opción "raw zone" se ven estos datos.
