Plan A

Para instala un sistema operativo libre en una mac tenemos que tener un CD o DVD vivo insertado en la mac y apagar el equipo, luego prenderlo y durante el inicio presionar la tecla Alt durante varios segundos, este debería darnos la opción de iniciar con otro disco.

Nota: Si tu mac es muy vieja puede ser que necesites una versión PPC de GNU

PPC

Para esto podemos usar Ubuntu, que tiene buen soporte para maquinas antiguas.

Intel

Si no lo hace podemos instalar rEFit que nos permitirá elegir el disco de inicio durante el inicio de la mac y volver a repetir el procedimiento. El resto es igual a Instalar GNU/Linux en cualquier tipo de equipos.


Plan B

Esta es una guía de pasos para usar software libre sobre OS X para los que no nos queda otra opción lo recomendable es usar el plan A y migrar a un sistema libre. Pero sin o podes hacer esto acá hay una lista de los programas disponibles con fácil instalación y también existen manejadores de paquetes (programas) similares a los de los sistemas libres como con Homebrew y Macports.

Arte

  • Gimp Similar a Photoshop pero libre (requiere X11 )
  • FFView Visualizador de imagenes y lector de historietas
  • Inkscape Similar a Ilustrator o Corel (requiere X11 )
  • MyPaint Excelente programa para dibujar con tableta de dibujo.
  • Pencil Programa simple para animaciones dibujas cuadro a cuadro en la PC
  • Blender Programa para animar (en 2D y 3D), modelar en 3D, editar videos, dibujar.
  • Audacity Editor de sonido
  • Baudline Imágenes del espectro sonoro en tiempo real.
  • PureData Programador gráfico. Similar a quartz composer pero libre
  • Reactivision Reconocimientos de TUIO y movimientos de los dedos, convirtiendolos en OSC y Midi. Excelente programa para integrar a PureData, Animata, Blender, etc
  • Animata Programa para hacer animación con huesos (no disponible en Macports)
  • jahshakaFX Editor de vídeo y efectos
  • HandBrake Conversor de DVD y vídeos a múltiples formatos.
  • macam Controlador para webcams en OSX. Luego de instalar este controlador se podrá usar casi cualquier webcam en macosx

Electronica

  • Fritzing Programa para realizar plaquetas y circuitos electrónicos en protoboard
  • Arduino Programación y electrónica para arduino y otros maquinas libres fácilmente programables.

Web y Red

  • Firefox Navegador web libre
    • Lector de PDF en firefox 3
  • Kompozer Editor de paginas web
  • NVU Compositor simple de paginas web.
  • Thunderbird Cliente de correos electronicos
  • Transmission Cliente de BitTorrent
  • CyberDuck Cliente de FTP
  • FileZila Cliente de FTP
  • Chicken of the VNC Visualizador de VNC
  • Vine VNC server Servidor de VNC
  • Fldigi Red sonora

Mensajería

  • Adium Mensajería Cifrada para Jabber, Gtalk, Skype, Facebook y más de modo cifrado. Podes mirar el tutorial de como usar IRC con Adium
  • Colloquy Cliente mensajería de IRC
  • Fire Mensajería IRC, Jabber, AIM, ICQ, Microsoft, Yahoo!,

Textos

  • LibreOffice Editor estilo Office pero libre
  • Abiword Editor estilo word simple y ligero
  • MacDJView Lector de documentos DjVu
  • FBreader Lector de Epub
  • Smultron Editor del texto plano.

Reproductores

  • Mplayer Un reproductor de video y audio muy bueno, mucho mejor que Quicktime
  • VLC Reproductor de video y audio, se pueden hacer listas de reproducción estilo itunes.
  • iTerm Terminal libre

Herramientas de sistema

  • TestDisk Reparación de discos y búsqueda de archivos borrados
  • JumpCut Historial para el porta papeles
  • Zipper Compresor y descompresor. Reemplazo del compresor del sistema.
  • Desktop Manager Multiples escritorios (Reemplazo libre de Spaces)
  • Brun OSX grabador de CDROM/DVD
  • Tor PPC x86 Red anónima y proxy anónimo para Internet.
  • Git PPC x86 Manejador de proyectos. Podes usar la interfaz GitX


Homebrew

Es un manejador de paquetes libres para OSX con el vamos a podes instalar algunas cosas que no se encuentra fácilmente.

Instalación

  • Descargar Xcode para tu versión de OS X tenes que generar una cuenta (Poner datos falsos)
  • Luego vas a tener que abrir la terminal (ir a Aplicaciones, luego abrir Utilidades y ahi se encuentra la Terminal) y escribir lo que corresponde a tu Mac:

Mac nuevas (10.6 en adelante)

ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"

Versiones antiguas( 10.5 y anteriores)

ruby -e "$(curl -fsSkL raw.github.com/mistydemeo/tigerbrew/go/install)"

y agregar en ~/.bash_profile 

export PATH=/usr/local/sbin:/usr/local/bin:$PATH

Uso

Buscar una aplicación

port search inkscape

Instalar una aplicación

brew install inkscape

Nota: La primera instalación va a tardar mucho, por que necesita de muchas dependencias no existentes en OSX.

Desintalar una aplicación

brew uninstall gimp-app

Actualizar la lista de programas

brew update

Actualizar todos los programas

brew update; brew upgrade

Macports

Instalación

  1. ir a MacPorts y descargar la instalación para tu versión de OS X
  2. descargar Xcode para tu versión de OS X tenes que generar una cuenta (Poner datos falsos)
  3. instalar Xcode y luego Macports.
  4. Ir a la aplicación terminal (en Aplicaciones -> Utilitarios) y poner sudo port -v selfupdate cuando solicite la clave introducirla, la clave no aparece en pantalla.

Uso

Buscar una aplicación

port search gimp

Instalar una aplicación

Con macports uno puede instalar una gran cantidad de aplicaciones libres, por ejemplo: 

sudo port install gimp-app
Nota: La primera instalación va a tardar mucho, por que necesita de muchas dependencias no existentes en OSX.

Desintalar una aplicación

sudo port uninstall gimp-app

Actualizar la lista de programas

sudo port selfupdate

Actualizar todos los programas

sudo port upgrade outdated

Correr demonios al inicio del sistema

sudo port load lighttpd

Iniciar demonios a mano

sudo launchctl start org.macports.lighttpd
