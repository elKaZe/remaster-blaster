Transmitir con VLC

Para esto elegimos Emitir (Ctrl+S)

Como fuente para la trasmisión podes elegir la cámara el microfono, un video, etc. 

Servidor de streaming: http://fm.hackcoop.com.ar
Número de puerto: 8000
Punto de montaje: /nuestraradio.ogg
Nombre de usuario: dejar vacio
Contraseña: hackme

{Tip|Comproba que la calidad sea para web (por ejemplo: ogg)}

El VLC se queda transmitiendo

Uno puede recibir la transmisión en cualquier navegador.

Desde la terminal

cvlc -vvv ~/Vídeos/video.ogv --sout '#transcode{vcodec=theo,vb=512,fps=10,scale=0.5,acodec=vorb,ab=32,channels=2,samplerate=11250}:std{access=shout,mux=ogg,dst=usuario:clave@fm.hackcoop.com.ar:8000/video.ogg}'
