Qué trae

  • Pidgin con OTR e IRC habilitados
  • Firefox con plugins de privacidad y seguridad habilitados
  • Thunderbird con enigmail instalado, para enviar y recibir mails cifrados
  • LibreVPN
  • Ciboulette
  • Tor y el Tor Browser Bundle para navegación anónima
  • Scantailor y herramientas de digitalización
  • IDJC para transmitir radio
  • Fish una terminal para principiantes
  • Zim una wiki de escritorio para documentar lo que aprendés en el hacklab
  • Mumble para participar en asambleas remotas/distribuidas
  • Transmission y Syncthing para compartir archivos
