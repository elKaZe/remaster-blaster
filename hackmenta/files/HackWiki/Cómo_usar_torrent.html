Cómo usar torrent

Primero vas a tener que instalar el Trasmission Windows, GNU/Linux o OS X y una vez instalado, abrirlo:

Nota: En Trisquel, Debian o Ubuntu: sudo aptitude install transmission-gtk

Buscar un torrent

Busca lo que quieras en Torrentz, este busca en varios sitios web a la vez y darle clic al archivo que queres descarga y elegí el sitio (ejemplo "The Pirate Bay") en el icono del magneto o descarga. En Mozilla Firefox debe aparecerá un menú como este: Tan solo elegí el transmission y aceptar

¡Ya esta descargando!

Nota:
  • Tal vez debas reiniciar el Mozilla Firefox antes de usar transimission por primera vez.
  • Además de Firefox este tutorial funciona con muchos navegadores, como: Midori, Icecat, Iceweasel, etc
  • The Pirate Bay no almacena películas solo archivos .torrent; y estos no funcionan sin transmission.
  • Si algo no baja muy rápido aprovecha para buscar más cosas, ¡¡no tenes limites de cantidad, ni peso!!
  • Para descargar y compartir; deja abierto el transmission la mayor parte del tiempo posible
