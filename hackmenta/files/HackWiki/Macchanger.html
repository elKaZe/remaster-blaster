Macchanger


Instalar

sudo apt-get install macchanger

Correr al inicio del sistema

Edita /etc/rc.local y agregar lo siguiente para cambiar el nombre de tu placa de red.

Puede hacerlo a través de la terminal nota: aquí lo estoy abriendo con el editor de texto "gedit", pero puedes usar otro. nota 2: sudo significa permisos especiales ya que es un archivo de configuración del sistema.

O puedes hacer esto a través del navegador de archivos

Una vez abierto el archivo:

Agregas: 

ifconfig | grep Link | cut -d " " -f1 | grep -v lo | while read RED; do
       /usr/bin/macchanger -r $RED
done

De este modo las redes no podrán reconocer tu equipo, cada vez que reiniciemos la computadora vamos a tener una mac address diferente, o sea un numero diferente de identificación de nuestro equipo de red.
