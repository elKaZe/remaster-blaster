Tipografías


Como instalas tipografías

En GNU/Linux hay muchos tipos de tipografía, algunas son especificas de la terminal y otras de los programas gráficos. Lo mejor es buscar tipografías libre, bajo licencias como las GPL, CC, o licencias personales que respeten la libertad Buscar tipografías

Hay muchos lugares donde buscar tipografías, las mas recomendables son las libres en el sistema y algunos sitios web podes encontrar muchas 

apt-cache search ttf

y van a aparecer todas las tipografías true type disponibles en tu distribución, lo siguiente es elegir algunas e instalarlas. 

apt-get install ttf-vlgothic ttf-unfonts

Y se pueden buscar tipografías en varios sitios web:

  • Font Squirrel que tiene muchas tipografías libres
  • Abstract Fonts tiene muchas tipografías libres
  • http://www.googlefonts.com/ Descargarlas no usarlas enlazadas a google :(
  • http://www.theleagueofmoveabletype.com
  • http://www.grsites.com/archive/fonts/
